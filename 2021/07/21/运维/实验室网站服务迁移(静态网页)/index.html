Github Hook配置

Github配置位置，项目的Settings->Webhooks->Add webhook。填写对应的信息

  • Payload URL: 域名+端口（flask服务配置）
  • Content type：json
  • Secret：自定义密钥，在脚本中配置
  • Which events would you like to trigger this webhook?：仅在推送时进行操作
